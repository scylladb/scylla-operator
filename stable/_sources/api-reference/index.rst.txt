API Reference
=============

All APIs provided by Scylla Operator are defined using CRDs and adhere to Kubernetes API standards. You can find out how Kubernetes style REST APIs work on `<https://kubernetes.io/docs/reference/using-api/>`_ and `<https://kubernetes.io/docs/reference/access-authn-authz/>`_.
